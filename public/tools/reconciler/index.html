🤖 Ollama — ذكاء اصطناعي محلي (RAG كامل)

الموديل شغّال على جهازك عن طريق Ollama. المساعد بيبعتله سؤالك + المقاطع المسترجعة من الفهرس ويرد عليك — مفيش أي داتا بتخرج برّا الجهاز.

—
الحل (مرة واحدة) — اسمح للصفحة تكلم Ollama:
1️⃣ افتح CMD وانسخ (دوس على الأمر ينسخ لوحده): setx OLLAMA_ORIGINS "*"
2️⃣ اقفل Ollama تمامًا: كليك يمين على أيقونته جنب الساعة → Quit. وللتأكيد افتح Task Manager واقفل أي ollama شغّال.
3️⃣ افتح Ollama تاني → ارجع هنا → اختبار الاتصال.
لو مستعجل من غير ريستارت دائم: CMD جديد → set OLLAMA_ORIGINS=* && ollama serve وسيبه مفتوح.
مفيش سيرفر شغّال على العنوان ده:
1️⃣ اتأكد إن Ollama مفتوح (أيقونته جنب الساعة)، أو شغّله من CMD: ollama serve
2️⃣ جرّب تفتح في المتصفح: http://localhost:11434 — المفروض تشوف "Ollama is running".
3️⃣ لو Ollama أصلًا على جهاز زميلك مش جهازك: localhost عمره ما هيوصّلك — اكتب IP جهازه: http://192.168.x.x:11434 (افتح "🌐 تشغيله للتيم" تحت).
بتحاول توصل لجهاز السيرفر ومش راضي — راجع على جهاز السيرفر نفسه:
1️⃣ Ollama سامع على الشبكة؟ لازم يكون اتعمل عليه: setx OLLAMA_HOST "0.0.0.0" + قفل Ollama تمامًا وفتحه تاني.
2️⃣ الفايروول فاتح للبورت؟ CMD كـ Administrator: netsh advfirewall firewall add rule name="Ollama LAN" dir=in action=allow protocol=TCP localport=11434
3️⃣ الاتنين على نفس الشبكة (نفس الواي فاي/السويتش)؟ جرّب من جهازك: ping 192.168.x.x
4️⃣ الـ IP صح ومتغيّرش؟ على السيرفر: ipconfig وشوف IPv4 الحالي.
🌐 تشغيل الـ AI للتيم كله — جهاز واحد يبقى سيرفر (مرة واحدة)
على الجهاز اللي عليه Ollama والموديلات (جهازك):
1️⃣ خليه يسمع للشبكة: CMD → setx OLLAMA_HOST "0.0.0.0" ثم اقفل Ollama من جنب الساعة تمامًا وافتحه.
2️⃣ افتح الفايروول: CMD كـ Administrator → netsh advfirewall firewall add rule name="Ollama LAN" dir=in action=allow protocol=TCP localport=11434
3️⃣ هات الـ IP بتاعك: ipconfig → خد IPv4 Address (مثال: 192.168.1.50).
4️⃣ كل واحد في التيم يفتح ⚙️ هنا ويكتب في العنوان: http://الـIP-بتاعك:11434 → اختبار الاتصال → يختار الموديلات → حفظ.
⚠️ مهم: جهازك لازم يفضل شغّال ومش بينام (Settings → Power → Sleep: Never وهو في الشاحن). ولو الـ IP بيتغير مع الراوتر، ثبّته (DHCP Reservation). لو التيم سأل في نفس اللحظة، الطلبات بتترتب ورا بعض — عادي لفريق صغير.

بتبدّل بينهم من زرار ⚡/🧠 فوق في الشات، أو ابدأ رسالتك بـ "فكر:" تجبر موديل التفكير لسؤال واحد. ترشيحات: سريع → qwen2.5:1.5b · تفكير → qwen2.5:7b أو deepseek-r1:7b.
